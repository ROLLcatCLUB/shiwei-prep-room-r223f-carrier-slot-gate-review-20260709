# R223D Bottom Input Non Chat Visual Policy

## Purpose

The bottom input exists only for local notes, D-option supplements, and material/class constraints. It is not a chat composer.

## Visual Policy

| Attribute | Required Static Direction |
| --- | --- |
| Size | compact, lower visual weight than prep canvas |
| Placeholder | local constraint wording, not open chat wording |
| Position | below workbench flow, not primary interaction center |
| Action label | add note / add local constraint, not send chat |
| Scope indicator | current section only |

## Allowed Placeholder Examples

```text
补充本环节材料限制...
添加本班情况说明...
给这个组件加一条本地要求...
```

## Forbidden Placeholder Examples

```text
问我任何问题...
告诉小教你想怎么改整份教案...
继续聊天生成...
```
